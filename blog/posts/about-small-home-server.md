# 家里有一台一直亮着的小服务器，生活会很不一样

![家用服务器](https://images.unsplash.com/photo-1558494949-ef010cbdcc31?auto=format&fit=crop&w=1600&q=80)

当照片、自动化、备份、智能家居和个人工具都开始往一个小服务器上聚拢，很多事情会突然变顺。它未必强大，但足够稳定、足够安静，也足够长期。

真正改变体验的不是“服务器”这三个字，而是很多以前分散的小事开始被安静地收拢起来：备份有去处，服务有位置，自动化也终于不再飘在各个设备之间。

# 家里有一台一直亮着的小服务器，生活会很不一样

![家用服务器示意](https://images.unsplash.com/photo-1518773553398-650c184e0bb3?auto=format&fit=crop&w=1600&q=80)

当照片、自动化、备份、智能家居和个人工具都开始往一个小服务器上聚拢，很多事情会突然变顺。它未必强大，但足够稳定、足够安静，也足够长期。

# 如何继续把它做成完整内容系统

![内容系统](https://images.unsplash.com/photo-1497366412874-3415097a27e7?auto=format&fit=crop&w=1600&q=80)

一个网站只发文章还不够，结构也很重要。

## 可以继续补的模块

- 分类 / 标签
- 置顶排序
- 专题页
- 年份归档
- 搜索优化
- RSS / 订阅
- 图片压缩与现代格式
- 自动部署与备份

## 这套站已经有的基础

- `photo/` 自动照片墙
- `blog/posts/` 自动文章生成
- 置顶文章
- 搜索框
- GitHub Actions 自动发布

## 下一步最值得做的

### 1. 分类
例如：
- 技术
- 生活
- 随笔
- 设备

### 2. 专题
例如把“建站”系列教程做成一个专门入口。

### 3. 归档
让旧内容更容易找回。

内容系统真正成熟起来，不在于页面多，而在于：

> 新内容进来以后，它会自动落到正确的位置。

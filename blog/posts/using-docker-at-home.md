# 家庭服务器为什么越来越适合用 Docker

![Docker 示意图](https://images.unsplash.com/photo-1605745341112-85968b19335b?auto=format&fit=crop&w=1600&q=80)

当服务一多，Docker 最大的价值不是“潮流”，而是把环境管理这件事一下子变清楚了。它会让你的服务迁移、备份和重建都省心很多。
